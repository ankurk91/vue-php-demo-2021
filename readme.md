# Vue.js v2 + php v8 demo

Sample project to work with Vue.js v2.x

### Requirements

* node.js v14.x || v16.x
* yarn v1.22 (classic)
* php v8.0
* composer v2.1

### Setup

* Clone the repo
* Install dependencies

```bash
cp .env.example .env
yarn install
composer install

```

* Run the project

```bash
yarn run hot
composer serve
```
