# Vue js + php demo

Sample project for Vue.js v2.x

### Requirements

* node.js v14.x
* yarn v1.22 (classic)
* php v8.0
* composer v2.1

### Setup

* Clone the repo
* Install deps

```bash
cp .env.example .env
yarn install
composer install

```

* Run the project

```
yarn run hot
composer serve
```
